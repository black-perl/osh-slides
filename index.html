What  are SBCs ?
A single-board computer (SBC) is a
complete computer built on a single
circuit board, with microprocessor(s),
memory, input/output (I/O) and other
features required of a functional computer.

Open Source hardware (OSH)
The information about the hardware is 
easily discerned so that others can also
make it.
Hardware design (i.e. mechanical drawings,
schematics, bills of material, PCB layout
data, HDL source code and integrated
circuit layout data), in addition to the
software that drives the hardware, are all
released under open source licenses.
3
OSS v/s OSH

"Free as in speech, not as in food"
Major differences between developing
open-source software and developing
open-source hardware is that hardware
results in tangible outputs, which cost money
to prototype and manufacture while
developing software happens to be a cake
walk.

Arduino

Pros : 

- Consumes very less power.
- Perfect for beginners.
- Insanely popular, so tons of tutorials available.
- Quite cheap  ~ $30.
- Lots of variants available.
- Well suited for interfacing with the real world
   objects.

9
Arduino
Cons:

- No GUI(Graphical User Interface).
- Suitable for only single purpose projects,
   unable to handle large number of
   processes/threads.



10
Raspberry Pi
11
Raspberry Pi
Pros : 

- Full power of a tiny computer.
- Can run any Linux distro from an external
   SD-card.
- Packed with a HDMI port, 2 USB ports and an 
   ethernet port. 
- Priced around  ~ $35.
- Easy for beginners.

12
Raspberry Pi
Cons : 

- Lacks interfacing support for external sensors
   and buttons.
- Arduino will be a good choice if you want 
  to interface your home electronics but if you
  want the power of a computer Raspberry Pi 
  would be the way to go.
13

Why BeagleBone ?

- Arduino + Raspberry Pi => BeagleBone

- It has the power of the Raspberry Pi, but it
   has the external interfacing options of the
   Arduino.

15

Pros : 

- 69 GPIO pins compared to the Raspberry Pi's
   8.
- Supports wide variety of OSes.
- Comes with a pre- installed OS in the inbuilt
   flash memory, so you can use it on the go.
- Only meant for serious developers.
- Priced around ~ 50$ - 200$.
